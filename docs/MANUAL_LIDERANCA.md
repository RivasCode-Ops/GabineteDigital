# Manual da Liderança — Gabinete Digital

## Perfil

Liderança comunitária (nível 20).

## Fluxos

### 1. Registrar Visita

```
Sidebar → Atividades → Nova Atividade
```

Campos:
- **Tipo**: Visita
- **Pessoas envolvidas**: selecionar contatos
- **Território**: local da visita
- **Data**: quando ocorreu
- **Descrição**: resumo do que foi tratado

### 2. Registrar Demanda

```
Sidebar → Demandas → Nova Demanda
```

Campos mínimos:
- **Título**: descrição resumida
- **Categoria**: saúde, educação, infraestrutura, transporte, etc.
- **Território**: local da demanda

> Acompanhe o status pelo sistema. A coordenação atualizará o andamento.

### 3. Consultar Agenda

```
Sidebar → Agenda
```

Visualizar:
- Próximos eventos
- Reuniões agendadas
- Atividades programadas

### 4. Registrar Contato

```
Sidebar → Pessoas → Nova Pessoa
```

Registrar novos contatos feitos durante visitas:
- Nome e telefone da pessoa
- Território onde foi abordada
- Observações relevantes

### 5. Participar de Pesquisas

```
Sidebar → Pesquisas
```

Registrar:
- Sentimento da comunidade (muito favorável a muito resistente)
- Perguntas e respostas coletadas
- Problemas identificados
- Prioridades da comunidade

## Perguntas Frequentes

**Preciso cadastrar toda pessoa que encontro?**
Sim. Isso alimenta a inteligência territorial e permite à coordenação conhecer a base.

**Quanto tempo leva para registrar uma visita?**
Cerca de 2 minutos. As informações mínimas são tipo, data e descrição.

**Posso registrar demanda de qualquer lugar?**
Sim. Use o sistema pelo celular ou computador.

**O que fazer se esquecer de registrar no momento?**
Registre assim que possível. O sistema aceita datas retroativas.

---

## Dicas

1. Registre as atividades **no mesmo dia** — dados recentes geram alertas mais precisos
2. Use as **categorias corretas** nas demandas — ajuda a coordenação a priorizar
3. Mantenha contatos **atualizados** — pessoas mudam de telefone
4. Se tiver dúvida, pergunte ao coordenador municipal
