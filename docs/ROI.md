# ROI — Gabinete Digital

## Quanto um gabinete perde sem o sistema?

### Cenário Real (antes do Gabinete Digital)

```
Item                        Horas/mês     Custo mensal
─────────────────────────────────────────────────────────
Organizar WhatsApp            30h         R$ 1.200
Planilhas de demanda          15h         R$ 600
Relatórios manuais            10h         R$ 400
Re-trabalho (perda de dado)   8h          R$ 320
Buscar informação perdida     12h         R$ 480
─────────────────────────────────────────────────────────
Total                         75h         R$ 3.000
```

**75 horas/mês** desperdiçadas com organização manual.

---

### Cenário com Gabinete Digital

```
Item                        Horas/mês     Custo mensal
─────────────────────────────────────────────────────────
Organizar WhatsApp            8h           R$ 320
Planilhas de demanda          2h           R$ 80
Relatórios automáticos (War Room) 0h       R$ 0
Re-trabalho                   1h           R$ 40
Buscar informação perdida     1h           R$ 40
─────────────────────────────────────────────────────────
Total                         12h          R$ 480
```

---

## Economia Direta

```
75h - 12h = 63 horas economizadas por mês

63h × R$ 40/hora (custo de coordenador) = R$ 2.520/mês

Custo da plataforma (Plano Gabinete): R$ 197/mês

ROI: R$ 2.520 / R$ 197 = 12,8x
```

**Cada R$ 1 investido retorna R$ 12,80.**

## ROI por Perfil

### Coordenador Geral

```
Antes: 4h/dia organizando informação
Depois: 30min/dia no War Room

Economia: 3,5h/dia × 20 dias = 70h/mês
```

### Coordenador Municipal

```
Antes: 2h/dia atualizando planilhas
Depois: 15min/dia registrando atividades

Economia: 1,75h/dia × 20 dias = 35h/mês
```

### Liderança

```
Antes: 30min/dia anotando e enviando WhatsApp
Depois: 5min/dia registrando no sistema

Economia: 25min/dia × 20 dias = 8h/mês
```
