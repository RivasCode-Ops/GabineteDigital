# Manual Rápido — Gabinete Digital

## Acessar

```
https://seudominio.com.br
Login: seu email
Senha: fornecida pelo admin
```

---

## Para Lideranças

### Registrar Visita

```
Atividades → Nova Atividade
Tipo: Visita
Pessoas: [selecionar contatos]
Território: [local]
Descrição: [o que foi tratado?]
```

> ⏱ 2 minutos.

### Registrar Demanda

```
Demandas → Nova Demanda
Título: [o que a comunidade precisa?]
Categoria: saúde, educação, infraestrutura...
Território: [local]
```

> Acompanhe o andamento pelo sistema.

### Registrar Contato

```
Pessoas → Nova Pessoa
Nome: [nome completo]
Telefone: [com DDD]
Território: [onde encontrou?]
```

> Sempre que conhecer alguém novo na comunidade.

### Ver Agenda

```
Agenda
```

> Veja reuniões, audiências e eventos marcados.

---

## Para Coordenadores Municipais

### Cadastrar Pessoa

```
Pessoas → Nova Pessoa
+ Nome, telefone, território, categoria
```

### Gerenciar Demandas da Região

```
Demandas → [filtro por território]
```

- Alterar status (Recebida → Triagem → Encaminhada → Acompanhamento → Resolvida)
- Acompanhar prazos

### Registrar Atividades da Equipe

```
Atividades → Nova Atividade
```

- Tipo: Visita, Reunião, Plantão, Evento, Mutirão
- Vincular pessoas participantes

### Participar de Pesquisas

```
Pesquisas → Nova Pesquisa
```

- Sentimento da comunidade
- Problemas identificados
- Prioridades

## Para Coordenadores Gerais

### Territórios

```
Territórios → Novo
Nome + Tipo + Território pai
```

### Lideranças

```
Lideranças → Nova
Pessoa + Função + Superior + Território
```

### War Room

```
War Room
```

- KPIs do gabinete
- Pipeline de demandas
- Alertas ativos
- Top territórios e lideranças

### Inteligência

```
Inteligência → Atualizar Indicadores
```

> Gera score (0–100) para cada território.

### Inteligência Artificial

```
IA
```

- Diagnóstico por território
- Insights gerais
- Report em texto

## Dicas

1. **Registre no mesmo dia** — dados frescos geram alertas precisos
2. **Use categorias corretas** — ajuda a coordenação a priorizar
3. **Cadastre toda pessoa que encontrar** — alimenta a inteligência territorial
4. **Deu dúvida?** Clique em "Encontrou dificuldade?" no canto inferior direito
5. **WhatsApp do grupo** — suporte rápido entre os usuários
