## FASE 01 — FUNDAÇÃO

### Objetivo

Construir a fundação técnica do projeto Gabinete Digital.

O sistema será uma plataforma SaaS multiusuário de gestão política, relacionamento comunitário, demandas, territórios e inteligência territorial.

Nesta fase não implementar regras de negócio.

Implementar apenas a infraestrutura base.

---

### Stack Obrigatória

Frontend:

* Next.js 15 App Router
* TypeScript
* Tailwind CSS
* Shadcn/UI
* React Hook Form
* Zod

Backend:

* API Routes Next.js
* Prisma ORM
* PostgreSQL

Infra:

* Docker
* Docker Compose

Autenticação:

* Auth.js
* JWT
* Refresh Token

---
